== Network Builder

Network Builder is a working prototype of the app that helps you to build networks on a map. App will find your current location with HTML5 geolocate and provide you an ability to CRUD network links on the real map.

Implemented with:
* Ruby 2
* Rails 4
* CoffeeScript
* HTML 5
* Google Maps API V3
* jQuery
* Underscore.JS
* Icons from VisualPharm
* etc

Tested in Chrome browser only.
